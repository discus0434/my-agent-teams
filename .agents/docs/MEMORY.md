# MEMORY

## Update Rules

- `.agents/docs/MEMORY.md` は lead だけが編集する。
- worker は `.agents/queue/memory_proposals/` に proposal を作る。
- 次回以降の作業で再利用できる教訓だけを残す。
- 1 entry = 1 lesson。
- source task / report / commit を必ず付ける。
- 追加前に既存 entry を検索し、重複する場合は更新または supersede する。
- 秘密情報、credential、個人情報、raw log、未検証の推測、一時的な進捗は書かない。
- 誤りを見つけた場合は訂正し、必要なら old entry を superseded に移す。

## Entry Format

```md
- M-YYYY-MM-DD-NNN [scope][active][source:<task/report/commit>]
  教訓を1つだけ書く。
```

Workers do not edit this file.
